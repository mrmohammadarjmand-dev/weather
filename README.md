# 🌤️ Weather App | اپلیکیشن هواشناسی

یک وب‌اپلیکیشن زیبا و ساده برای نمایش آب‌وهوای شهرها با **نقشه تعاملی** و **نمودار پیش‌بینی ۲۴ ساعته**  
پشتیبانی کامل از **نام شهرهای فارسی** و انگلیسی 🕌

## ✨ امکانات
- 🔍 جستجوی شهر با نام فارسی یا انگلیسی (مثلاً "اردبیل" یا "Tehran")
- 🌡️ نمایش دما، رطوبت، سرعت باد و دمای احساسی
- 🗺️ موقعیت شهر روی نقشه با مارکر (Leaflet.js)
- 📈 نمودار دمای ۲۴ ساعت آینده (Chart.js)
- 🕒 تاریخچه جستجوهای اخیر (ذخیره در مرورگر)
- 🌙 حالت تاریک / روشن

---

## 🛠️ تکنولوژی‌ها
- HTML5, CSS3, JavaScript (Vanilla)
- [Leaflet.js](https://leafletjs.com/) - نقشه
- [Chart.js](https://www.chartjs.org/) - نمودار
- [OpenStreetMap Nominatim](https://nominatim.org/) - جستجوی موقعیت
- [OpenWeatherMap API](https://openweathermap.org/api) - داده‌های هواشناسی
